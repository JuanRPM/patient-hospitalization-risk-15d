# patient-hospitalization-risk-15d

Machine learning model to predict patient hospitalization within 15 days based on clinical and contextual data.

## Project Structure

- `data/raw/` – synthetic input dataset (CSV).
- `data/processed/` – cleaned and feature-engineered datasets.
- `notebooks/` – exploratory analysis and modeling notebooks.
- `src/data/` – scripts to generate and load data.
- `src/models/` – scripts to train and evaluate models.
- `reports/` – figures and summary reports.
- `dashboards/` – BI dashboards (e.g. Power BI).

More details and results will be added as the project evolves.
